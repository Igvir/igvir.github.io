# Product Overview

Personal portfolio website for Igvir Ramirez, a Cloud Solutions Architect & Software Architect.

- **Live URL**: https://www.igvir.com
- **Purpose**: Showcase AWS certifications, professional experience, technical expertise, blog posts, and LinkedIn activity
- **Audience**: Recruiters, potential clients, professional network
- **Languages**: English (default at `/`) and Spanish (`/es/`)

## Key Features

- Responsive single-page layout with fixed sidebar navigation
- Dark mode with system preference detection and localStorage persistence
- Offline support via Service Worker (PWA)
- Contact form with reCAPTCHA v3 and Formspree backend
- Embedded LinkedIn posts and blog article cards
- SEO optimized (Open Graph, Twitter Cards, JSON-LD structured data, hreflang, sitemap)
- WCAG AA accessibility compliant

## Hosting

- GitHub Pages from `main` branch
- Custom domain via CNAME (`www.igvir.com`)
- HTTPS enforced with HSTS
